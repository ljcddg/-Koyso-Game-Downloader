# Koyso 游戏下载器

基于 Python Tkinter 的图形化游戏下载工具，支持浏览、搜索和下载 Koyso 平台的游戏资源。

## 功能特性

- 🎮 **游戏浏览**：支持按分类浏览全部游戏（动作、冒险、射击、RPG等15种分类）
- 🔍 **多条件搜索**：支持按游戏名称关键词搜索
- 📊 **排序功能**：支持按热门程度和最新发布排序
- 📄 **分页展示**：支持页码跳转和上下页切换
- 🖼️ **图片预览**：异步加载游戏封面图片
- ℹ️ **详情查看**：弹窗显示游戏详细介绍、版本号和文件大小
- ⬇️ **一键下载**：自动获取下载链接并在浏览器中打开
- 🎨 **美观界面**：现代化的 UI 设计，支持悬停效果

## 技术栈

- **Python 3.x**
- **Tkinter**：图形界面框架
- **Requests**：HTTP 请求库
- **BeautifulSoup4**：HTML 解析
- **Pillow**：图片处理
- **Threading**：多线程异步加载

## 使用方法

### 1. 运行程序

### 2. 基本操作

- **选择分类**：从下拉菜单选择游戏类型
- **输入关键词**：在搜索框输入游戏名称
- **切换排序**：选择"热门"或"最新"
- **查看详情**：点击"查看"按钮浏览游戏介绍
- **下载游戏**：点击"下载"按钮获取下载链接

### 3. 分页导航

- 使用"上一页"/"下一页"按钮翻页
- 在"跳至"输入框输入页码后按回车或点击"GO"

## 核心功能说明

### 游戏列表加载
- 启动时自动加载最新游戏
- 支持动态切换分类和排序方式
- 异步加载游戏详情和图片，避免界面卡顿

### 游戏详情弹窗
- 单例模式设计，同一时间只显示一个详情窗口
- 支持在不关闭窗口的情况下切换查看不同游戏
- 异步加载游戏介绍文本和高清封面图

### 下载功能
- 通过 API 获取加密的下载链接
- 自动在默认浏览器中打开下载页面
- 状态栏实时显示操作进度

## 注意事项

- 程序需要网络连接才能正常使用
- 部分游戏可能需要登录后查看详细信息
- 下载链接由第三方平台提供，请遵守相关使用条款
